## Title

LED Video Screening Tool for Urban LED Display Screen Light-Pollution Exposure

## Creators

- Bing Zhang
- Yi Lin

## Description

This repository provides a supplementary Python tool for video-based screening of urban LED display screens. The tool supports polygon annotation of target screens, extraction of video-derived luminance-proxy, temporal, blue-channel, screen-background contrast, and chromaticity-occupancy indicators, and generation of screening flags and dashboard summaries. It is intended for preliminary screening and priority identification rather than calibrated photometric, spectroradiometric, or health-risk assessment.
